# Task Manager API 🚀

API RESTful para gerenciamento de tarefas, desenvolvida com Node.js, Express e PostgreSQL.

Este projeto foi criado com o objetivo de consolidar conhecimentos em:
- Estruturação de APIs
- CRUD completo
- Integração com banco relacional (PostgreSQL)
- Organização de projeto backend
- Boas práticas de desenvolvimento

---

## 📌 Tecnologias Utilizadas

- Node.js
- Express
- PostgreSQL
- pg (node-postgres)
- dotenv
- cors
